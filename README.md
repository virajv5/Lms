#LMS

=>An Library Management System, which is made to get the basic task done easily
such as

*Create a new User


*Fetch all the Users


*Add new book


*Fetch all the books


*Fetch specific book


*Delete a book 


*Borrow a book 


*Return a book

=>The Project Backend is developed using Spring Boot, Hibernate, Mysql DataBase.


=>For Frontend React JS is used.
